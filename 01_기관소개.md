# 기관 소개

**한국국제협력단**(영문: Korea International Cooporation Agency, **KOICA**)은 대한민국의 대외 무상협력 사업을 전담하는 위탁집행형 준정부기관입니다.

| 항목 | 내용 |
|------|------|
| 기관명 | 한국국제협력단 (KOICA) |
| 설립일 | 1995년 4월 1일 |
| 설립 근거 | 「한국국제협력단법」 |
| 소속 | 기획재정부 |
| 본사 | 서울특별시 종로구 |

한국국제협력단은 대한민국과 개발도상국 간의 우호협력관계 및 상호교류를 증진하고, 후진국의 경제·사회 발전을 지원함으로써 국제협력 증진에 기여하는 것을 목적으로 설립되었습니다.
